# FChat

![docs/images/]

Simple chat application using ExpressJS and Socket.io

Try [demo](http://fchat.cakapcakap.com) 

## How to use

1. Clone this repository.

  ```bash
  git clone https://github.com
  ```

2. Install the dependencies. 
  
  ```
  npm install
  ```

3. Start the app.

  ```
  node bin/www
  ```
